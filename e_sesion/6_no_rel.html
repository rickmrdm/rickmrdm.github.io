Diseño no Relacional

Replit nos proporciona una base de datos tipo par llave-valor, donde cada registro tiene una llave y un valor asociado.

La llave identifica el registro y debe ser texto.

El valor puede ser de cualquier tipo de literal, incluyendo literales de objeto.

Para asignar valor a una llave se utiliza el método async set; por ejemplo: 

await bd.set("pepito", "cuentos");

Para recuperar el valor asignado a una llave se utiliza el método async get; por ejemplo: 

let pass = await bd.get("pepito");

Para este proyecto, la llave es el nombre de usuario y el valor asociado es la contraseña encriptada.
