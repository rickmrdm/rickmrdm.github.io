Instrucciones

  1. Crea un sitio de Replit de nombre aut con Node.js y base de datos.

    Las instrucción para generar el archivo b.js es:

    echo module.exports = \"$REPLIT_DB_URL\" > b.js

    Las instrucciones para agregar librerías son:

    npm i express
    npm i @replit/database
    npm i multer

  2. Edita el código, utilizando las páginas que siguen.

  3. Prueba tu código localmente con Visual Studio Code.

  4. Súbelo a Replit y pruébalo.
