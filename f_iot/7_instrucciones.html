Instrucciones

Aquí no se muestran videos, porque los procesos son como los de la sección Control de sesión.

  1. Crea un sitio de Replit de nombre iot con Node.js y base de datos.

    Crea el archivo b.js desde la pestaña Shell de Replit con el comando: 

    echo module.exports = \"$REPLIT_DB_URL\" > b.js

    Las instrucciones para agregar librerías son:

    npm i express
    npm i @replit/database
    npm i cors
    npm i uuid-apikey

  2. Edita el código, utilizando las páginas que siguen.

  3. Prueba tu código localmente con Visual Studio Code.

  4. Súbelo a Replit y pruébalo.
