Instrucciones

  1. Crea la base de datos iot en Atlas y agrega un documento por cada dispositivo.

  2. Crea un sitio de Replit de nombre iot con Node.js. El proceso es parecido al de la sección Control de sesión. La instrucción para agregar librerías es: npm i express cors mongodb

  3. Edita el código, utilizando las páginas que siguen.

  4. Prueba tu código localmente con Visual Studio Code. El proceso es parecido al de la sección de Control de sesión.

  5. Súbelo a Replit y pruébalo. El proceso es parecido al de la sección de Control de sesión.
